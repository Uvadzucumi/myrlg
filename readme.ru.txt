--- Управление ---

	стрелки - перемещение персонажа
	o - открыть дверь
	i - инвентарь
	    a-zA-Z - выбрать предмет инвентаря.
		    w - одеть/взять в руки
			d - бросить (не реализовано)
			r - прочитать
    space - включить/выключить диффузное освещение         		
	esc - выход
	
--- Настройки ---

Настройки меняются путем редактирования значений параметров файла myrlg.conf. Можно задать:

  Разрешение окна по умолчини:
screen_width: 800		// ширина
screen_height: 600      // высота
screen_bpp: 32          // глубина цвета
full_screen: 0          // 1 - полноэкранный режим, 0 - оконный

    Скорость перемещения персонажа:
movement_delay: 50		// задержка между перемещением персонажа в новую ячейку карты уровня в милисекундах

autoopen_doors: 1       // 1 - включает автоокрытие дверей при перемещении персонажа 

# Light options
see_only_with_light: 1      // 1 - персонаж видит только освещенные ячейки карты, 0 - все что попадает в FOV
enable_diffuse_light: 1     // 1 - включает распространение отраженного света
